## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ or Bun
- npm, yarn, or bun

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AbuAbdULLAH-MuhammadAli/portfolio.git
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   # Using bun (recommended)
   bun install

   # Or using npm
   npm install

   # Or using yarn
   yarn install
   ```

3. Start the development server:
   ```bash
   # Using bun
   bun dev

   # Or using npm
   npm run dev

   # Or using yarn
   yarn dev
   ```

4. Open your browser and navigate to `http://localhost:5173`
